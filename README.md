```
转换一个虚线，点，下划线，空格字符串到大驼峰 foo-bar Foo-Bar
```

```

```

- transliteration
```
用于nodejs,浏览器,web worker等的utf-8到ascll音译模块

```

- markdown-it-container
```
给markdown-it的markdown解析器创建的块级自定义容器插件
```

```

```

- fileSave
```
当需要你保存比blob的尺寸限制还大或者没有足够的ram的巨大文件,使用新的stream api来异步保存数据到硬盘驱动上
```
